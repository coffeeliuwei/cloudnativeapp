## 一、实训目的

1. 理解微服务架构与云原生"本地做一步、云上换一步"的改造主线；
2. 掌握将本地微服务项目（coffee 订单/快递系统）部署到阿里云的完整流程（RDS、MSE Nacos、EDAS）；
3. 了解 CI/CD 流水线、外网访问与网络、测试运维、安全加固等生产化能力。

**内容**：coffee 微服务项目的本地运行 → 阿里云资源准备 → 平移上云 → EDAS 部署上云，以及 CI/CD、网络、运维、安全等提高内容。
